Movimentação de colunas

Você pode mover a coluna apenas informando a classe .mov-? onde ? é o número desejado, veja o exemplo a baixo:

.col-2
.col-2 .move-1
.col-2 .move-2
.col-2 .move-3
.col-6 .move-4
.col-2 .move-10
Fixar colunas em dispositivos móveis

Para que a coluna tenha seu tamanho fixo mesmo em dispositivos móvies, adicione a classe .columns n div pai das colunas

.row .column

.col-4
.col-4
.col-4

O mesmo é necessário para que movimentação de colunas aconteça no mobile

.row .column

.col-4 .move-4
Tamanhos de colunas diferentes de acordo com o dispositivo

É possível definir tamanhos diferentes para diferentes resoluções, caso defina .col-2 no desktop as colunas irão equivaler a 1/6 das 12 colunas, automaticamente em dispositivos móveis a coluna ocupara 100% do tamanho, porem em alguns casos se faz necessário forçar o tamanho de apenas uma ou mais colunas, neste caso juntamente com o .col-2 podemos passar uma nova classe para o dispositivo em questão, por exemplo: .col-2 .tab-4 .mob-6. Desta forma em desktop teremos 6 colunas ocupando 2 espaços totalizando 12 colunas, no tablet teremos 6 colunas ocupando quatros espaços e em smartphones teremos 6 colunas ocupando 6 espaços.

.col-2 .tab-4 .mob-6

.col-2 .tab-4 .mob-6
Exibir ou ocultar elementos em diferentes resoluções

Adicionando a classe .hide-desk o elemento é ocultado apenas no desktop, .hidden-tab o elemento é ocultado apenas no tablet e .hidden-mob o elemento é ocultado apenas em smartphones.

.hidden-desk Oculto para Desktop
.hidden-tab Oculto para Tablet
.hidden-mob Oculto para Smartphone

Adicionando a classe .show-desk o elemento será exibido apenas no desktop, .show-tab o elemento será exibido apenas no tablet e .show-mob o elemento será exibido apenas em smartphones.

.show-desk Exibido para Desktop
.show-tab Exibido para Tablet
.show-mob Exibido para Smartphone
